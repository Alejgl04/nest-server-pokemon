# Execute a development

1. Clone the repository
2. Execute
```
 yarn install
```
3. Nest cli installed
```
  npm i -g @nestjs/cli
```
4. Execute database

5. Rebuild database with seed
```
  localhost:3000/api/v2/seed
```

```
docker-compose up -d
```
